# Appendix A: Key Benchmarks Explained

| Benchmark | What It Measures | Saturation (2026) | Reference |
|-----------|-----------------|-------------------|-----------|
| **MMLU** | Multi-task language understanding across 57 subjects | ~96% | [Paper](https://arxiv.org/abs/2009.03300) |
| **GSM8K** | Grade-school math word problems | ~98% | [Paper](https://arxiv.org/abs/2110.14168) |
| **MATH** | Competition mathematics | ~94% | [Paper](https://arxiv.org/abs/2103.03874) |
| **SWE-bench Verified** | Real-world GitHub issue resolution | ~97% | [Homepage](https://www.swebench.com/) |
| **HumanEval** | Python code generation from docstrings | ~95%+ | [GitHub](https://github.com/openai/human-eval) |
| **BIG-Bench** | 204 diverse tasks (reasoning, logic, common sense) | Mixed | [Paper](https://arxiv.org/abs/2206.04615) |
| **HellaSwag** | Commonsense reasoning (sentence completion) | ~95% | [Paper](https://arxiv.org/abs/1905.07830) |
| **TruthfulQA** | Truthfulness and avoiding common misconceptions | ~70% | [Paper](https://arxiv.org/abs/2109.07958) |
| **MMMU** | Multimodal understanding (college-level) | ~89% | [Homepage](https://mmmu-benchmark.github.io/) |
| **MathVista** | Visual math reasoning | ~82% | [Homepage](https://mathvista.github.io/) |
| **OSWorld** | Real computer task completion by agents | ~66% | [Paper](https://arxiv.org/abs/2404.07972) |
| **Chatbot Arena** | Human preference ranking (ELO) | No cap | [Leaderboard](https://lmarena.ai/) |

**Key insight:** Most benchmarks are saturating. New, harder benchmarks are needed to track continued progress.
