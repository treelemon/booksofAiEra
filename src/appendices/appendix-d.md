# Appendix D: Glossary of Key Terms

| Term | Definition |
|------|------------|
| **AGI** | Artificial General Intelligence — AI that matches human cognitive ability across all domains |
| **Alignment** | Ensuring AI systems pursue the goals their creators intend |
| **Attention** | The mechanism that lets transformers weigh the importance of different input tokens |
| **Chain-of-thought** | Prompting technique where the model generates intermediate reasoning steps |
| **Diffusion model** | Generative model that learns to denoise random noise into coherent data |
| **Emergence** | Capabilities that appear at scale without explicit training |
| **Fine-tuning** | Further training a pre-trained model on a specific task |
| **Foundation model** | Large model trained on broad data, adaptable to many tasks |
| **GPU** | Graphics Processing Unit — the primary hardware for training neural networks |
| **Hallucination** | When AI generates confident falsehoods |
| **In-context learning** | Model learning to perform a task from examples in the prompt (no weight updates) |
| **Jagged frontier** | The irregular boundary where AI is superhuman on some tasks and subhuman on others |
| **Jailbreak** | A prompt designed to bypass a model's safety guardrails |
| **LLM** | Large Language Model |
| **MoE** | Mixture of Experts — architecture that activates only subset of parameters per token |
| **RLHF** | Reinforcement Learning from Human Feedback — alignment technique |
| **Reasoning model** | Model trained to generate extended chains of thought before answering |
| **RAG** | Retrieval-Augmented Generation — grounding generation in retrieved documents |
| **SWE-bench** | Benchmark that tests AI on real-world GitHub issue resolution |
| **Sycophancy** | Tendency of AI to agree with user's stated or implied position |
| **Test-time compute** | Inference-time computation (thinking) that improves answer quality |
| **Transformer** | The neural network architecture underlying most modern LLMs (Vaswani et al., 2017) |
