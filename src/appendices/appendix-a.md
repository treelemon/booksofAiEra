# Appendix A: Key Benchmarks Explained

| Benchmark | What It Measures | Saturation (2026) |
|-----------|-----------------|-------------------|
| **MMLU** | Multi-task language understanding across 57 subjects | ~96% |
| **GSM8K** | Grade-school math word problems | ~98% |
| **MATH** | Competition mathematics | ~94% |
| **SWE-bench Verified** | Real-world GitHub issue resolution | ~97% |
| **HumanEval** | Python code generation from docstrings | ~95%+ |
| **BIG-Bench** | 204 diverse tasks (reasoning, logic, common sense) | Mixed |
| **HellaSwag** | Commonsense reasoning (sentence completion) | ~95% |
| **TruthfulQA** | Truthfulness and avoiding common misconceptions | ~70% |
| **MMMU** | Multimodal understanding (college-level) | ~89% |
| **MathVista** | Visual math reasoning | ~82% |
| **OSWorld** | Real computer task completion by agents | ~66% |
| **Chatbot Arena** | Human preference ranking (ELO) | No cap |

**Key insight:** Most benchmarks are saturating. New, harder benchmarks are needed to track continued progress.
