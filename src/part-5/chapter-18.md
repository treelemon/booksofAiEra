# Chapter 18: The Path to AGI

## 18.1 Defining AGI

There is no consensus definition. Common criteria:
- **Economic:** AI that can perform any economically valuable work at human level
- **Cognitive:** AI that matches or exceeds human performance on all cognitive tasks
- **Generality:** AI that can learn any task from scratch, like a human

Srinivas's "AGI is just the last tool you ever need to build" captures the pragmatic view.

## 18.2 Current Capability Assessment

Using Morris et al.'s "Levels of AGI" framework:

| Level | Description | Examples | Progress |
|-------|-------------|----------|----------|
| 0 (No AI) | Purely mechanical | Calculator | N/A |
| 1 (Emerging) | Narrow, equal to/exceeds unskilled human | GPT-3, BERT | Surpassed |
| 2 (Competent) | Narrow, >50th percentile skilled adult | GPT-4, o1 | In progress |
| 3 (Expert) | Narrow, >90th percentile skilled adult | o3, DeepSeek R1 | Partial |
| 4 (Virtuoso) | Narrow, >99th percentile | AlphaFold | Few domains |
| 5 (Superhuman) | Narrow, surpasses best humans | AlphaGo | Few domains |

**Key insight:** Different systems reach different levels on different tasks. No system is Level 3+ across all domains.

## 18.3 Paths to AGI

| Path | Proponents | Argument |
|------|-----------|----------|
| Scale current approach | Scaling optimists | More data + compute + parameters → AGI |
| Reasoning models | OpenAI, DeepSeek | Test-time compute unlocks general reasoning |
| Agents + architectures | Many labs | Adding memory, tools, planning bridges gaps |
| New architectures | Academic researchers | Transformers hit a wall; new paradigm needed |
| Embodied learning | Robotics labs | Grounding in physical world is necessary |
| Neuro-symbolic | Hybrid researchers | Combine neural learning with symbolic reasoning |

**My view:** A combination of scaling, reasoning, and agentic architectures will likely produce something that *looks like* AGI within 3–10 years. Whether it *is* AGI depends on your definition.

## 18.4 Signs to Watch

- **Self-improvement:** AI that can improve its own capabilities
- **Transfer learning:** One model excels across diverse, unrelated tasks
- **Meta-learning:** AI that learns how to learn new tasks
- **Theory of mind:** AI that models other agents' mental states
- **Situation awareness:** AI that understands its own circumstances

Any of these showing rapid improvement would suggest AGI is closer than expected.
