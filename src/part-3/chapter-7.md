# Chapter 7: Agentic AI — The 2026 Breakthrough

## 7.1 What Is an AI Agent?

An AI agent is a model that can:
- **Perceive** its environment (via tools, APIs, files)
- **Plan** multi-step sequences of actions
- **Use tools** (code execution, web search, databases, APIs)
- **Remember** context across interactions
- **Act autonomously** toward a goal

## 7.2 The 2025–2026 Leap

| Capability | Early 2025 | Mid 2026 |
|-----------|-----------|----------|
| OSWorld (computer tasks) | 12% | 66% |
| Web navigation | 15% | ~50% |
| Multi-step tool use | Fragile | Reliable |
| Long-running tasks | Minutes | Hours |

## 7.3 Architecture of an Agent

- **Model:** the reasoning engine (the "brain")
- **Tool library:** pre-built capabilities (search, code, APIs)
- **Memory:** short-term (context) + long-term (vector store)
- **Planning loop:** observe → reason → act → observe
- **Human-in-the-loop:** escalation for uncertainty

## 7.4 Current Limitations

- **Reliability:** 66% success means 1 in 3 tasks fail
- **Cost:** agent loops are expensive (many API calls per task)
- **Security:** prompt injection, tool misuse, data leakage
- **Evaluation:** no standardized benchmark for agent performance
- **Patience:** user tolerance for agent failures is low

## 7.5 The Promise

When agents cross the 90%+ reliability threshold, the implications are vast:
- Automated software engineering
- Personal AI assistants that truly act on your behalf
- Enterprise automation of entire business processes
- Scientific lab automation

Agentic AI is the most important trend of 2026.
