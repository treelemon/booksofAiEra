# Appendix D: Glossary of Key Terms

| Term | Definition | Reference |
|------|------------|-----------|
| **AGI** | Artificial General Intelligence — AI that matches human cognitive ability across all domains | [Wikipedia](https://en.wikipedia.org/wiki/Artificial_general_intelligence) |
| **Alignment** | Ensuring AI systems pursue the goals their creators intend | [Wikipedia](https://en.wikipedia.org/wiki/AI_alignment) |
| **Attention** | The mechanism that lets transformers weigh the importance of different input tokens | [Wikipedia](https://en.wikipedia.org/wiki/Attention_(machine_learning)) |
| **Chain-of-thought** | Prompting technique where the model generates intermediate reasoning steps | [Paper](https://arxiv.org/abs/2201.11903) |
| **Diffusion model** | Generative model that learns to denoise random noise into coherent data | [Wikipedia](https://en.wikipedia.org/wiki/Diffusion_model) |
| **Emergence** | Capabilities that appear at scale without explicit training | [Wikipedia](https://en.wikipedia.org/wiki/Emergence) |
| **Fine-tuning** | Further training a pre-trained model on a specific task | [Wikipedia](https://en.wikipedia.org/wiki/Fine-tuning_(deep_learning)) |
| **Foundation model** | Large model trained on broad data, adaptable to many tasks | [Wikipedia](https://en.wikipedia.org/wiki/Foundation_model) |
| **GPU** | Graphics Processing Unit — the primary hardware for training neural networks | [Wikipedia](https://en.wikipedia.org/wiki/Graphics_processing_unit) |
| **Hallucination** | When AI generates confident falsehoods | [Wikipedia](https://en.wikipedia.org/wiki/Hallucination_(artificial_intelligence)) |
| **In-context learning** | Model learning to perform a task from examples in the prompt (no weight updates) | [Wikipedia](https://en.wikipedia.org/wiki/In-context_learning) |
| **Jagged frontier** | The irregular boundary where AI is superhuman on some tasks and subhuman on others | [HBR](https://hbr.org/2024/11/what-is-the-jagged-frontier-in-ai) |
| **Jailbreak** | A prompt designed to bypass a model's safety guardrails | [Wikipedia](https://en.wikipedia.org/wiki/Prompt_jailbreaking) |
| **LLM** | Large Language Model | [Wikipedia](https://en.wikipedia.org/wiki/Large_language_model) |
| **MoE** | Mixture of Experts — architecture that activates only subset of parameters per token | [Wikipedia](https://en.wikipedia.org/wiki/Mixture_of_experts) |
| **RLHF** | Reinforcement Learning from Human Feedback — alignment technique | [Wikipedia](https://en.wikipedia.org/wiki/Reinforcement_learning_from_human_feedback) |
| **Reasoning model** | Model trained to generate extended chains of thought before answering | [Paper](https://arxiv.org/abs/2408.03314) |
| **RAG** | Retrieval-Augmented Generation — grounding generation in retrieved documents | [Wikipedia](https://en.wikipedia.org/wiki/Retrieval-augmented_generation) |
| **SWE-bench** | Benchmark that tests AI on real-world GitHub issue resolution | [Homepage](https://www.swebench.com/) |
| **Sycophancy** | Tendency of AI to agree with user's stated or implied position | [Paper](https://arxiv.org/abs/2308.03958) |
| **Test-time compute** | Inference-time computation (thinking) that improves answer quality | [Paper](https://arxiv.org/abs/2408.03314) |
| **Transformer** | The neural network architecture underlying most modern LLMs (Vaswani et al., 2017) | [Paper](https://arxiv.org/abs/1706.03762) |
