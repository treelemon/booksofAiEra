# Appendix B: Timeline of Major AI Events (1956–2026)

## The Founding Era (1956–1980)

| Year | Event | Significance |
|------|-------|-------------|
| 1956 | Dartmouth Conference | AI officially named as a field |
| 1958 | Perceptron | First neural network |
| 1966 | ELIZA | First chatbot |
| 1969 | Shakey the Robot | First general-purpose mobile robot |
| 1974 | First AI Winter begins | Funding dries up after Lighthill report |
| 1976 | MYCIN | Expert system for medical diagnosis |

## The Expert Systems Era (1980–2000)

| Year | Event | Significance |
|------|-------|-------------|
| 1980 | XCON | First commercially successful expert system |
| 1987 | Second AI Winter | Expert systems fail to scale |
| 1995 | Support Vector Machines | Major ML breakthrough |
| 1997 | DeepBlue beats Kasparov | AI masters chess |
| 1997 | LSTM introduced | Long Short-Term Memory networks |
| 2000 | Statistical NLP takes over | Shift from rules to data |

## The Deep Learning Revolution (2006–2020)

| Year | Event | Significance |
|------|-------|-------------|
| 2006 | Deep Learning rebranded | Hinton's "Deep Belief Networks" |
| 2009 | ImageNet dataset launched | 14M labeled images |
| 2012 | AlexNet wins ImageNet | Deep learning breakthrough |
| 2013 | Word2vec | Word embeddings change NLP |
| 2014 | GANs introduced | Generative Adversarial Networks |
| 2014 | Seq2Seq + Attention | Neural machine translation |
| 2016 | AlphaGo beats Lee Sedol | AI masters Go |
| 2017 | "Attention Is All You Need" | Transformer architecture |
| 2018 | BERT | Bidirectional pre-training for NLP |
| 2018 | GPT-1 | Generative pre-training |

## The Foundation Model Era (2020–Present)

| Year | Event | Significance |
|------|-------|-------------|
| 2020 | GPT-3 (175B parameters) | In-context learning at scale |
| 2020 | AlphaFold 2 | Protein folding solved |
| 2021 | DALL-E | Text-to-image generation |
| 2022 | Stable Diffusion | Open-source image generation |
| 2022 | ChatGPT launched | 100M users in 2 months |
| 2023 | GPT-4 | Multimodal, improved reasoning |
| 2023 | Open source (Llama, Falcon, Mistral) | Democratization begins |
| 2023 | Bletchley Declaration | First international AI safety agreement |
| 2024 | OpenAI o1 | First reasoning model |
| 2024 | Sora | Text-to-video generation |
| 2024 | Claude 3, Gemini 1.5 | Frontier competition intensifies |
| 2025 | DeepSeek R1 | Open-source reasoning, cost efficiency |
| 2025 | o3, Claude 4, Gemini 2 | Reasoning models become standard |
| 2025 | AlphaEvolve | AI discovers novel algorithms |
| 2025 | SWE-bench hits ~100% | Coding benchmark saturation |
| 2025 | EU AI Act comes into force | Comprehensive regulation |
| 2025 | Stargate announced | $500B AI infrastructure |
| 2025 | OpenAI releases first open model | Competitive pressure from DeepSeek |
| 2026 | Agentic AI breakthrough | OSWorld 66% |
| 2026 | AI adoption reaches 88% (org) / 53% (population) | Fastest adoption in history |
| 2026 | US–China gap closes to 2.7% | True multipolar AI landscape |
