# 附录 D：关键术语表

| 术语 | 定义 |
|------|------|
| **AGI** | 通用人工智能——在所有领域匹配人类认知能力的 AI |
| **对齐** | 确保 AI 系统追求其创造者意图的目标 |
| **注意力机制** | 让 Transformer 权衡不同输入 token 重要性的机制 |
| **思维链** | 提示技术，模型生成中间推理步骤 |
| **扩散模型** | 学习将随机噪声去噪为连贯数据的生成模型 |
| **涌现** | 在特定规模下出现的未经显式训练的能力 |
| **微调** | 在预训练模型上针对特定任务进一步训练 |
| **基础模型** | 在广泛数据上训练、可适应多种任务的大型模型 |
| **GPU** | 图形处理器——训练神经网络的主要硬件 |
| **幻觉** | AI 自信地生成虚假事实 |
| **上下文学习** | 模型从提示中的示例学习执行任务（无权重更新） |
| **锯齿形前沿** | AI 在某些任务上超人类、在其他任务上亚人类的不规则边界 |
| **越狱** | 旨在绕过模型安全护栏的提示 |
| **LLM** | 大型语言模型 |
| **MoE** | 混合专家——每个 token 仅激活部分参数的架构 |
| **RLHF** | 基于人类反馈的强化学习——对齐技术 |
| **推理模型** | 训练后在回答前生成扩展思维链的模型 |
| **RAG** | 检索增强生成——将生成建立在检索文档上的技术 |
| **SWE-bench** | 测试 AI 在真实 GitHub 问题解决上的基准 |
| **谄媚** | AI 倾向于同意用户明示或暗示的立场 |
| **测试时计算** | 提高回答质量的推理时计算（思考） |
| **Transformer** | 大多数现代 LLM 的神经网络架构（Vaswani 等，2017） |
